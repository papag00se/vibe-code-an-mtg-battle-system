# Lesson 1 — Bootstrap Untap

This is it: your very first move. By the end of this lesson you'll have a real React app running on your machine, with your name on it (well, with *Untap* on it). Don't worry if you've never done this before — that's exactly who this is for. Paste the prompt, watch it work, and we'll explain everything after.

## Teaches

- Codex setup
- Vite app creation
- React entrypoint
- project standards
- footer disclaimer

## Codex prompt

```text
Goal:
Create a beginner-friendly React + TypeScript app called Untap.

Context:
This is a free personal vibe-coding tutorial for learning programming through Magic: The Gathering rules concepts. It is unofficial fan content, not endorsed by Wizards of the Coast.

Constraints:
Use Vite, React, TypeScript, and plain CSS.
Use Scryfall for card data later, but do not call it yet.
Do not use Wizards logos.
Do not claim official affiliation.
Do not build a commercial game client.
Keep the code beginner-readable.
Explain what you build after each step.

Done when:
The app runs locally.
The home screen says "Untap".
The footer includes the unofficial fan-content notice from AGENTS.md.
A static two-player battlefield placeholder is visible.
```

## Manual test

Time to see it live. Run these two commands:

```bash
npm install
npm run dev
```

Open the local Vite URL it prints, and confirm the title, footer, and battlefield placeholder all show up. If they do — congratulations, Untap is officially alive. That little battlefield is where everything else gets built.

## Ask Codex after implementation

Don't just admire it — ask how it works. Paste this and read the answer:

```text
Explain the project structure like I am new to coding. Explain main.tsx, App.tsx, components, and CSS.
```
