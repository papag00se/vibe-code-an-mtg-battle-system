# Lesson 1 — Bootstrap Stack Tutor

## Teaches

- Codex setup
- Vite app creation
- React entrypoint
- project standards
- footer disclaimer

## Codex prompt

```text
Goal:
Create a beginner-friendly React + TypeScript app called Stack Tutor.

Context:
This is a free personal vibe-coding tutorial for learning programming through Magic: The Gathering rules concepts. It is unofficial fan content, not endorsed by Wizards of the Coast.

Constraints:
Use Vite, React, TypeScript, and plain CSS.
Use Scryfall for card data later, but do not call it yet.
Do not use Wizards logos.
Do not claim official affiliation.
Do not build a commercial game client.
Keep the code beginner-readable.
Explain what you build after each step.

Done when:
The app runs locally.
The home screen says "Stack Tutor".
The footer includes the unofficial fan-content notice from AGENTS.md.
A static two-player battlefield placeholder is visible.
```

## Manual test

```bash
npm install
npm run dev
```

Open the local Vite URL. Confirm the title, footer, and battlefield placeholder appear.

## Ask Codex after implementation

```text
Explain the project structure like I am new to coding. Explain main.tsx, App.tsx, components, and CSS.
```
