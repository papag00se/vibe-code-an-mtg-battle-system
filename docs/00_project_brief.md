# 00 — Project brief

## Name

```text
Tutorial:  Build an MtG Battle System with AI!
The app:   Untap
```

## One-line description

Untap is a free, unofficial MtG rules-learning app you build with an AI agent — picking up real coding skills through card search, zones, turn phases, mana, combat, stack resolution, priority, and tests along the way.

## Audience

A Magic: The Gathering player who's curious about coding, likes how rules systems click together, and wants to build something real with Codex. Total beginners welcome.

## Educational goal

By the end, the learner should understand:

| Coding concept | MTG-shaped lesson |
|---|---|
| API calls | Search real cards through Scryfall |
| TypeScript types | Model cards, players, zones, and game state |
| React state | Life totals, hand, battlefield, stack UI |
| Arrays | Library, hand, graveyard, exile |
| State machines | Turn phases and combat steps |
| LIFO structures | Stack resolution |
| Validation | Legal vs illegal actions |
| Testing | Rules-engine unit tests |
| UX | Highlights, disabled actions, event log |

## Product goal

Build a local, free tutorial/sandbox. It should make MTG rules feel like software engineering concepts.

## Non-goals

Do not build:

- an MTG Arena replacement
- public matchmaking
- paid access
- a tournament platform
- a proxy/card-printing workflow
- an offline card-image archive
- a production-grade rules engine covering every corner case

## Guiding principle

Teach the shape of the rules first. Add precision later.
