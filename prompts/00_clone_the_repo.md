# Prompt 00 — Clone the repository

This is your true opening move. You don't have the project on your machine yet — so let your AI agent go get it. You're reading this on the web (or on GitHub); you don't need to have cloned anything to start.

Paste this into Codex (or any terminal-capable AI agent):

```text
Clone https://github.com/papag00se/vibe-code-an-mtg-battle-system and open the folder so we can start building.

Then read README.md, AGENTS.md, and docs/02_codex_and_ai_foundations.md, and give me a two-sentence summary of what we're about to build together.
```

Prefer to do it by hand? Same thing, in a terminal:

```bash
git clone https://github.com/papag00se/vibe-code-an-mtg-battle-system.git
cd vibe-code-an-mtg-battle-system
```

## Done when

- The repository is cloned to your machine.
- Codex (or your AI agent) is running inside the project folder.
- You've skimmed `README.md` and `docs/02_codex_and_ai_foundations.md` so you know the lay of the land.
